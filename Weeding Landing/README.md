# Весільне запрошення — Соломія & Назар 💍

Лендінг-запрошення: `index.html` — один файл, нічого встановлювати не треба.

## Telegram-група (замість RSVP)

Остання секція — запрошення в спільну Telegram-групу весілля: до свята там
питання й побажання, після — фото та враження від гостей.

У секції є QR-код (`tg-qr.svg`) і кнопка «Приєднатися до групи». Обидва ведуть
на посилання-запрошення групи. Щоб оновити лінк:
- кнопка: атрибут `href` у `<a id="tgBtn">` в `index.html`;
- QR: перегенерувати з нового лінка (`python3 -c "import segno; segno.make('ПОСИЛАННЯ').save('tg-qr.svg', scale=10, border=0)"`).

## Як опублікувати (Netlify, безкоштовно)

1. Зайди на [netlify.com](https://app.netlify.com) → зареєструйся.
2. Розділ **Sites** → перетягни мишкою папку `Weeding Landing` у вікно браузера.
3. Через хвилину сайт онлайн за адресою типу `random-name.netlify.app`.
4. У **Site settings → Domain management** можна змінити назву на щось типу
   `solomiia-nazar.netlify.app` або підключити власний домен.

## Фото ресторану

Секції «Локація» та «Атмосфера місця» об'єднані в одну: назва → фото → карта → маршрут.
Три фото ресторану вже завантажені й вставлені (`photos/shepit-1…3.jpg`, промо-фото
від студії Lofty). Щоб замінити на власні — поклади знімки з тими ж назвами
(деталі в `photos/README.txt`).

## Анімації та оформлення

- Hero (імена, дата) з'являються послідовно при завантаженні сторінки.
- Текст кожної секції «випливає» по черзі при прокрутці.
- Навколо секцій — декоративні оливкові рамки з кутовими гілочками.
- Якщо в системі ввімкнено «зменшити рух», анімації автоматично вимикаються.
